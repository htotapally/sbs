1. Create a public repository on GitHub where you will make this project accessible.
  2. Create a web application using JAX-RS (Jersey or RESTEasy) and MongoDB with the following features :
    1. Create a CRUD REST service to perform the following tasks
      1. Create – Create an employee
      2. Retrieve – Return one employee by ID or all employees
      3. Update – Update an employee
      4. Delete – Delete an employee
    2. Create a search REST service that returns employees that match a given last name
    3. Employee records should include the following fields :
      1. First Name
      2. Last Name
      3. Middle Initial
      4. Email Address
      5. Phone Number
      6. Position Category (Indirect, Direct, Program Manager, Director, Executive)
      7. Date Hired
      8. Address 1
      9. Address 2
      10. City
      11. State
      12. Zip Code
      13. Active Flag
  3. The application should build with maven.
  4. Create unit tests for the application using JUnit and your choice of mocking framework as needed.
  5. Once completed, push to GitHub and notify Jason Rock with the URL of the repository
